# Knowledge Sources

Drop files here for AI ingestion into the Clinical Signal knowledge base.

**Any format works:** PDF, Word, PowerPoint, Excel, text files, audio transcripts, even screenshots of marked-up labs.

## What to add

- **Course materials** — slide decks, student handouts, certification materials, decision trees
- **Clinical reference docs** — lab interpretation guides, supplement dosing charts, protocol templates, functional ranges
- **Call transcripts** — coaching call transcripts, case study walkthroughs, Q&A sessions (text format — we can't process audio/video directly)
- **Research articles** — papers, textbooks, or authors you reference in teaching

## What happens next

1. Files dropped here get extracted into structured knowledge entries
2. A review brief is generated with targeted clinical judgment questions
3. Dr. Laura reviews in a periodic 30-60 min session
4. Her answers become the highest-confidence entries in the knowledge base
